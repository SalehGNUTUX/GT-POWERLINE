# GT-POWERLINE

**GT-POWERLINE**

هو برنامج نصي بلغة باش يقوم بتثبيت Powerline و Fonts-Powerline وإعدادات الطرفية الخاصة بـ Bash و Zsh.

## الميزات
- يكتشف تلقائيًا نظام التشغيل ومدير الحزم.
- يتحقق مما إذا كانت Powerline مثبتة بالفعل، ويطلب إذن المستخدم للتثبيت من جديد إذا كانت مثبتة.
- يدعم Bash و Zsh.

## كيفية الاستخدام

يمكنك تشغيل الأداة مباشرة من المستودع باستخدام `curl` أو `wget`:

### باستخدام curl:
```bash
bash <(curl -s https://raw.githubusercontent.com/SalehGNUTUX/GT-POWERLINE/main/gt-powerline.sh)
```

### باستخدام wget:
```bash
bash <(wget -O - https://raw.githubusercontent.com/SalehGNUTUX/GT-POWERLINE/main/gt-powerline.sh)
```

## المتطلبات
- Bash أو Zsh
- مدير الحزم المناسب (apt أو pacman أو dnf أو yum)

## كيفية تثبيت الأداة من المستودع

إذا كنت تفضل تثبيت الأداة من المستودع، يمكنك اتباع الخطوات التالية:

1. **استنساخ المستودع**:

   يمكنك استنساخ المستودع باستخدام Git:

   ```bash
   git clone https://github.com/SalehGNUTUX/GT-POWERLINE.git
   cd GT-POWERLINE
   ```

2. **اجعل البرنامج النصي قابلاً للتنفيذ**:

   قم بجعل البرنامج النصي قابلاً للتنفيذ باستخدام الأمر التالي:

   ```bash
   chmod +x gt-powerline.sh
   ```

3. **تشغيل البرنامج النصي**:

   لتثبيت Powerline والإعدادات، قم بتشغيل البرنامج النصي باستخدام الأمر:

   ```bash
   ./gt-powerline.sh
   ```
